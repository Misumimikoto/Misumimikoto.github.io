关于题目“德才论”的个人理解

想看原题戳这里

核心思想

本题看起来信息量比较大，但是核心要掌握好。一大段文字描述其实表明了一件事：首先将这些人的成绩按照一定的标准分类，并且按照一定的顺序排列不同类的人。而同一类的人中，又按照一定的规则继续排列，因此很明显：这道题分类主要分为两个内容，首先将所有人分为五类，并且按照一定的顺序进行分类。然后同一类的人按照总分降序徳分降序准考证号升序。

注意点

1. 本题应该建立关于学生的结构体，存储一定的信息

struct student{
    int cai,de;//才分和徳分
    int flag;//分类
    char id[10];//准考证号
};

其中 分类 flag 这个参数尤其重要

2. 比较的标准必须是首先将学生按照分类来排序，之后才是总分降序，徳分降序。同时，在按照分类的过程中。必须首先将不及格的人排除掉，否则之后会将不及格的人和已经筛选出来的三类人混在一起

具体的分类代码如下

if(stu[i].de < L || stu[i].cai < L>){
  stu[i].flag = 5;//归为第五类人
  m--;//输出总人数减一
}
else if(stu[i].de >= H && stu[i].cai >= H) stu[i],flag = 1;
else if(stu[i].de >= H && stu[i].cai < H) stu[i].flag = 2;
else if(stu[i].de > stu[i].cai) stu[i].flag = 3;
else stu[i].flag = 4;

此处体现了一个很重要的思路：对于题目暗示的可以分类的数据，可以先给他在输入之时分好类 (赋值flag) ，再继续进行排序，这样就可以有条理的输出了。
